# Magic Stone · 原石狂磨

> 收集 · 打磨 · 交易 — 一款矿石打磨收集类移动端游戏

## 游戏简介

你是一名矿石工匠，从最普通的**原石**开始，通过打磨将其一步步升级为**玛瑙 → 翡翠 → 钻石**。在商城和黑市中低买高卖，完成任务获取奖励，最终集齐图鉴中所有珍稀矿石。

---

## 核心玩法

### 🔨 打磨系统

选择矿石和工具进行打磨，每次打磨有概率触发**升级**。等级越高，升级概率越低：

| 等级跨越 | 基础成功率 | 传奇工具加成 |
|---------|-----------|------------|
| 原石 → 玛瑙 | 50% | 65% |
| 玛瑙 → 翡翠 | 30% | 45% |
| 翡翠 → 钻石 | 15% | 30% |

升级到翡翠或钻石时会随机获得**子等级**，最高子等级即为顶级矿石：

| 子等级 | 概率 |
|--------|------|
| 1 级 | 43% |
| 2 级 | 30% |
| 3 级 | 20% |
| **4 级（顶级）** | **7%** |

打磨需要消耗游戏币，费用随等级递增：原石免费 → 玛瑙 300 → 翡翠 800。

每次打磨会增加矿石**损耗**，损耗达到上限后无法继续打磨。工具也会消耗**耐久**，耐久耗尽后需要购买新工具或合成升级。

### 💰 经济系统

#### 商城
- 每 **30 分钟**自动刷新，可花 **1,500 币**立即刷新
- 每次刷新出现 **6 件商品**（必定包含原石和玛瑙各一个）
- 固定价格购买，支持 **购买和出售**
- 出售价格为买价的 **70%**，磨损越高卖价越低

#### 黑市商人
- 每 **1 小时**自动刷新，可花 **20,000 币**立即刷新
- 收购和出售**独立随机**，同一物品两边价格可能不同
- 价格在商城买价的 **60% ~ 200%** 之间浮动
- 需要自己判断是商人出价高还是商城卖价高
- 每人限购一件，购买后**售罄**

### 🔧 工具合成

将 **3 个同等级工具**合成为 **1 个更高等级工具**：

| 工具等级 | 损耗系数 | 耐久消耗 | 升级加成 |
|---------|---------|---------|---------|
| 普通 | ×1.0 | -5/次 | +0% |
| 专业 | ×0.8 | -4/次 | +5% |
| 顶级 | ×0.5 | -2/次 | +10% |
| 传奇 | ×0.2 | -1/次 | +15% |

### 📖 矿石图鉴

共 **10 种**矿石可收集，一旦获得即**永久解锁**（即使之后卖掉也会保留记录）。每种矿石显示市场参考价，方便判断交易时机。

### 📋 任务系统

| 类型 | 说明 |
|------|------|
| **日常任务** | 每日凌晨 2:00 刷新。打磨、购买、出售等基础操作均可获得奖励 |
| **成就任务** | 一次性，达成条件永久完成 |
| **寻宝任务** | 集齐**帝王绿翡翠、蓝钻和非洲之心**三件顶级矿石，奖励 **20,000 币** |

## 构建指南

技术栈：React 18 + TypeScript + Vite + TailwindCSS + Capacitor

### 开发

```bash
npm install
npm run dev          # 浏览器访问 http://localhost:3000
```

### Android 构建

```bash
npm run build        # 构建 web 资源到 dist/
npm run cap:sync     # 同步到 Android 项目
npx cap open android # 在 Android Studio 中打开
```

然后在 Android Studio 中 Build → Run 即可。
